# 🌐 Study Group Finder  
Study Group Finder is a full‑stack MERN application designed to help students connect, create, and join study groups easily. It provides a modern interface and backend support to manage groups, members, and sessions.

---

## 🚀 Features  
- 📝 Create and manage study groups  
- 👥 Join or leave groups  
- 📅 Schedule study sessions  
- 📊 Dashboard with group details  
- 🔍 Search and filter groups by subject  
- 🔒 Secure authentication system  

---

## 🛠️ Tech Stack  
- MongoDB  
- Express.js  
- React.js  
- Node.js  
- Axios

## ⚙️ How to Run  
1. Download or clone the repository  
2. Navigate to backend folder → run `npm install` and `npm start`  
3. Navigate to frontend folder → run `npm install` and `npm start`  
4. Open **http://localhost:3000** in your browser
