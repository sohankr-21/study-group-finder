# Study Group Finder (MERN)

## 📖 Description
Study Group Finder is a MERN full‑stack app that helps students connect with peers who share the same subject and preferred study time.  
- **Frontend**: React form for user signup  
- **Backend**: Node.js + Express routes  
- **Database**: MongoDB Atlas with Mongoose  

## 🚀 Features
- User signup with name, subject, and time  
- Data stored in MongoDB Atlas (`study-group-finder` database)  
- Backend routes for authentication and group matching  
- End‑to‑end flow: React → Express → MongoDB Atlas
